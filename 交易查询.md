# 交易查询

交易查询，或称链码查询。本章节重点分析msp身份认证过程，及查询数据过程。

流程：
1. 客户端，发起链码查询 (client -> peer)
2. peer查出本地数据，响应返回结果给客户端 (peer -> client)

